## Random
![random](https://github.com/heewhy/Edu_CSharp/assets/55123531/139461b6-2051-4c2c-8889-20ac1d2ca4da)
C# 프로젝트를 생성

### 요구 사항
- 랜덤한 정수 3번을 줄바꿈하여 출력
- 첫번째 정수는 1 ~ 10 사이의 숫자를 출력
- 두번째 정수는 100 ~ 123 사이의 숫자를 출력
- 세번째 정수는 54 ~ 66 사이의 숫자를 출력
### Input
- 없음
### OutPut
- 숫자(1 ~ 10)
- 숫자(100 ~ 123)
- 숫자(54 ~ 66)
